# garden

#### 介绍
springboot应用
1.用户，角色管理。登录无Session，使用JWT认证。
2.菜单管理。
3.功能管理。
4.组织管理。
5.Lookup，数据字典管理。
6.Quartz定时任务调度。
7.Redis缓存，Redis分布式锁。
8.文件上传管理。
9.动态freemarker。
10.国际化，基于数据库。

#### 软件架构
软件架构说明
后端
应用：SpringBoot.2.0.2.RELEASE
数据库：MySql5.7
数据库连接池：hikari
缓存：Redis
模板框架：freemarker

前端
UI框架：Layui
JS工具：JQuery
